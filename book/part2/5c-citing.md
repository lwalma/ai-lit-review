# Citing Sources

A very important step in writing a literature review (or any scientific paper), is to correctly refer to the sources that you used. After all, you have to give credits to whom they are due, especially in academia.

## Recap Standard Practice: Citing Sources

When you are working with a limited number of sources, you should consult the guidelines of the particular citation style and format your references accordingly, both in the text and in the reference list. You can use tools, such as this <a href="https://www.tudelft.nl/tulib/apa-citation-examples" target=_blank>APA Citation Table</a> from TU Delft or the <a href="[url](https://www.scribbr.nl/bronvermelding/generator/)" target="_blank">reference generator</a> from Scribbr, to help you format your references correctly.

If you are working with a large number of sources, it is efficient to use a reference manager to manage and cite your sources in the format of your choice. Have a look at this video to learn more about how to use these tools:

```{warning}
Carefully check if your reference list includes **all** sources that you refer to in the text (and no sources you do not refer to). Also make sure to always carefully read the source before citing it to make sure you do justice to its content.
```

## Why You Should Not Let AI 'Improve' Your Reference List

Although it may be tempting to ask a GenAI tool to format your reference list, you should **not** do that. There are ample examples in which students as well as professional researchers used AI and ended up with <a href="https://delta.tudelft.nl/en/article/number-of-ghost-references-in-scientific-articles-growing-rapidly" target="_blank">'ghost references'</a> (non-existing sources) in their reference list. So do **not** ask AI to find additional sources for you, do **not** ask it to create a reference description for a particular source you found, do **not** ask it to improve your reference list, do **not** even ask it to put the sources in the correct order. All of these actions oftentimes do more harm than good.

It may not be the most exiting part of your research project, but you should compile a correct reference list yourself. If you want to work more efficiently, you can use a reference manager. If you are working in Overleaf, you should make sure to define your reference style correctly and add all sources correctly to your bibtex, and then you will get a correct reference list.
